# Buscador de usuarios

Este proyecto consiste en crear una mini-app que permita buscar usuario/as y mostrar sus datos en pantalla, consumiendo una API pública.

https://jsonplaceholder.typicode.com/users

![Input para escribir un nombre y botón para buscar usuario/as. Con una tarjeta de color verde con los datos del resultado](buscador-users.png)
