# Generador de chistes

Este mini proyecto consiste en un generador de chistes.

He usado la API de Chuck Norris:
**https://api.chucknorris.io/jokes/random**

## Mejoras

- Resetear el texto del botón cuando empieza a cargar
- Evitar que el loader se vea demasiado poco tiempo
- Añadir un pequeño efecto visual al chiste
