# Temporizador en JavaScript

## Ejercicio con setTimeout

1. Crea un botón en tu HTML.
2. Cuando el usuario haga clic en ese botón, debe aparecer un mensaje en pantalla (o en la consola) pero después de 3 segundos de espera.
3. El mensaje puede ser algo como "¡Hola!" o "Acción completada".
4. Opcional: añade otro botón para cancelar la acción antes de que pasen los 3 segundos.

## Ejercicio con setInterval

1. Crea un contador que empiece en 0.
2. Cada segundo, ese contador debe aumentar en 1 automáticamente y mostrarse en pantalla.
3. Cuando el contador llegue a 10, debe detenerse solo.
4. Opcional: añade un botón para reiniciar el contador desde 0.
